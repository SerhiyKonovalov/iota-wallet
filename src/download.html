Installation Guide

Step-by-step instructions for installing IOTA Wallet on your system

1

Download the Installer

Click the "Download for Windows" button above to download the IOTA-Wallet-Setup.exe file to your computer.

2

Run the Installer

Double-click the downloaded file and follow the installation wizard. You may need to allow the app through Windows Defender SmartScreen.

3

Launch IOTA Wallet

Once installed, launch IOTA Wallet from your Start menu or desktop shortcut. The first launch may take a few moments to initialize.

4

Create or Import Wallet

Follow the setup wizard to create a new wallet or import an existing one using your seed phrase. Make sure to backup your seed phrase securely.

1

Download the DMG File

Click the "Download for macOS" button to download the IOTA-Wallet.dmg file to your Downloads folder.

2

Mount and Install

Double-click the DMG file to mount it, then drag the IOTA Wallet app to your Applications folder.

3

Allow App Permissions

When first launching, you may need to allow the app in System Preferences > Security & Privacy if macOS blocks it.

4

Setup Your Wallet

Complete the initial setup by creating a new wallet or importing an existing one. Enable Touch ID for enhanced security if available.

1

Download AppImage

Download the IOTA-Wallet.AppImage file. This is a portable format that works on most Linux distributions.

2

Make Executable

Open terminal and run: chmod +x IOTA-Wallet.AppImage to make the file executable.

3

Run the Application

Double-click the AppImage file or run it from terminal: ./IOTA-Wallet.AppImage

4

Optional: Desktop Integration

For desktop integration, you can use AppImageLauncher or manually create a .desktop file in ~/.local/share/applications/
