Introduction to IOTA Wallet

Welcome to the official IOTA Wallet documentation. This guide will help you get started with managing your IOTA tokens securely and efficiently.

What is IOTA Wallet?

IOTA Wallet is a secure, user-friendly application designed to help you manage your IOTA tokens. Built with quantum-resistant security and zero-fee transactions, it provides the most advanced features for the IOTA ecosystem.

Key Features

  • Quantum-resistant cryptographic security
  • Zero-fee transactions on the IOTA network
  • Built-in staking functionality
  • Multi-platform support (Desktop, Mobile, Web)
  • Hardware wallet integration
  • Real-time network monitoring
Note: IOTA Wallet supports both IOTA and Shimmer networks, allowing you to manage multiple assets from a single interface.

Installation

Follow these steps to install IOTA Wallet on your device.

Desktop Installation

Windows

  1. Download the Windows installer from the official website
  2. Run the downloaded .exe file
  3. Follow the installation wizard
  4. Launch IOTA Wallet from the Start menu

macOS

  1. Download the macOS installer from the official website
  2. Open the downloaded .dmg file
  3. Drag IOTA Wallet to your Applications folder
  4. Launch IOTA Wallet from Applications
macOS Users: You may need to approve the app in System Preferences > Security & Privacy if you see a security warning.

Linux

  1. Download the Linux package for your distribution
  2. For Ubuntu/Debian, install the .deb package:
bash 
sudo dpkg -i iota-wallet_2.1.4_amd64.deb

Mobile Installation

iOS

  1. Open the App Store on your iPhone or iPad
  2. Search for "IOTA Wallet"
  3. Tap "Get" to download and install
  4. Open the app to begin setup

Android

  1. Open Google Play Store on your Android device
  2. Search for "IOTA Wallet"
  3. Tap "Install" to download the app
  4. Open the app to begin setup

Quick Start Guide

Get up and running with IOTA Wallet in just a few minutes.

First Launch

When you first open IOTA Wallet, you'll be presented with two options:

Create New Wallet

Start fresh with a brand new wallet and seed phrase.

Restore Existing Wallet

Import your wallet using your recovery phrase or backup file.

Creating Your First Wallet

  1. Click "Create New Wallet"
  2. Choose a strong password to encrypt your wallet
  3. Write down your 24-word recovery phrase and store it safely
  4. Verify your recovery phrase by entering the requested words
  5. Your wallet is now ready to use
Critical: Never share your recovery phrase with anyone. Write it down on paper and store it in a secure location. This phrase is the only way to recover your wallet if you lose access.

Creating a Wallet

Learn how to create and configure your IOTA wallet.

Wallet Setup Process

Creating a new wallet involves several important steps to ensure the security of your funds.

Step 1: Password Creation

Your password encrypts your wallet data locally on your device. Choose a strong password that:

  • Is at least 12 characters long
  • Contains uppercase and lowercase letters
  • Includes numbers and special characters
  • Is unique and not used elsewhere

Step 2: Recovery Phrase Generation

Your recovery phrase consists of 24 randomly generated words. This phrase is the master key to your wallet and can restore access to your funds on any device.

Example Recovery Phrase 
abandon ability able about above absent absorb
abstract absurd abuse access accident account
accuse achieve acid acoustic acquire across
act action actor actress actual adapt add

Step 3: Wallet Name and Profile

Give your wallet a memorable name to help identify it if you manage multiple wallets. You can also customize:

  • Wallet color theme
  • Currency display preferences
  • Network settings (Mainnet/Testnet)

Multi-Account Support

IOTA Wallet supports multiple accounts within a single wallet. Each account:

  • Has its own unique address
  • Can be named for easy identification
  • Shares the same recovery phrase
  • Can have different balances

Sending Tokens

Learn how to send IOTA tokens to other addresses.

How to Send Tokens

  1. Open your wallet and ensure you're connected to the network
  2. Click the "Send" button in your account dashboard
  3. Enter the recipient's IOTA address
  4. Specify the amount to send
  5. Add an optional message (up to 256 characters)
  6. Review the transaction details
  7. Confirm and sign the transaction

Address Verification

Always verify the recipient address carefully. IOTA addresses are typically 63 characters long and start with "iota1".

Example IOTA Address 
iota1qpg2xkj66wwgn8p2ggnp7p582gj8g5r5xhmnsdw7rqkqwjpr9t6xhzfvqdx

Transaction Fees

One of IOTA's main advantages is zero transaction fees. You can send any amount without worrying about network fees.

Zero Fees: IOTA transactions are completely feeless, making it perfect for micropayments and frequent transactions.

Transaction Status

After sending, you can track your transaction status:

  • Pending: Transaction is being processed by the network
  • Confirmed: Transaction has been accepted by the network
  • Failed: Transaction could not be completed

Receiving Tokens

Learn how to receive IOTA tokens in your wallet.

Your Receive Address

To receive IOTA tokens, you need to share your address with the sender.

How to Find Your Address

  1. Open your wallet and select the account
  2. Click the "Receive" button
  3. Your address will be displayed with a QR code
  4. Share the address or QR code with the sender

Address Reuse

Unlike older cryptocurrency systems, IOTA addresses can be safely reused multiple times. You can use the same address for all incoming transactions.

Multiple Addresses

For privacy and organizational purposes, you can generate multiple receiving addresses within each account. This is useful for:

  • Separating different income sources
  • Tracking specific payments
  • Enhanced privacy

QR Code Sharing

The QR code feature makes it easy to receive payments in person. The sender can simply scan your QR code with their mobile wallet to automatically fill in your address.

Tip: You can also include a specific amount in the QR code to streamline payment requests.

Staking

Earn rewards by staking your IOTA tokens and helping secure the network.

What is Staking?

Staking allows you to lock your IOTA tokens to support network validation and earn rewards. Your staked tokens help secure the IOTA network while generating passive income.

Staking Benefits

  • Earn rewards on your holdings
  • Support network security and decentralization
  • Participate in network governance
  • Flexible staking periods

How to Stake

  1. Navigate to the "Staking" section in your wallet
  2. Select the amount you want to stake
  3. Choose your staking period (30, 60, 90, or 180 days)
  4. Review the estimated rewards
  5. Confirm your staking transaction

Staking Periods and Rewards

Period Annual Yield Lock Duration Early Unlock
30 days 5% APY 30 days Available with penalty
60 days 7% APY 60 days Available with penalty
90 days 10% APY 90 days Available with penalty
180 days 15% APY 180 days Not available
Important: Staked tokens are locked for the chosen period. Early unstaking may result in reduced or no rewards.

Seed Management

Learn how to properly manage and secure your wallet seed phrase.

Understanding Your Seed Phrase

Your seed phrase (also called recovery phrase or mnemonic) is a 24-word sequence that represents your wallet's private keys. It is the master key to your wallet and provides complete access to all your funds.

Seed Security Rules

Never:
  • Share your seed phrase with anyone
  • Store it digitally (photos, cloud, email)
  • Enter it on any website
  • Share it with support teams

Best Practices for Seed Storage

  • Write it down on paper using a pen
  • Store in a fireproof and waterproof safe
  • Consider using a metal backup device
  • Keep multiple copies in separate secure locations
  • Never store on any device connected to the internet

Verifying Your Seed

Regularly verify that you have correctly recorded your seed phrase:

  1. Go to Settings > Security > Verify Recovery Phrase
  2. Enter the requested words from your backup
  3. Confirm that your backup is correct

Backup & Recovery

Learn how to backup and recover your wallet.

Creating a Backup

Regular backups ensure you can always recover your wallet. IOTA Wallet offers several backup options:

Stronghold Backup File

Stronghold is IOTA's encrypted backup system. To create a backup:

  1. Navigate to Settings > Backup
  2. Click "Create Stronghold Backup"
  3. Choose a secure location to save the file
  4. Set a strong backup password
  5. Store the backup file securely

Recovery Phrase Backup

Your 24-word recovery phrase is the most important backup. With this phrase, you can restore your wallet on any device, even if all other backups are lost.

Restoring Your Wallet

From Recovery Phrase

  1. Open IOTA Wallet and select "Restore Wallet"
  2. Choose "Restore from Recovery Phrase"
  3. Enter your 24-word recovery phrase in order
  4. Create a new password for this device
  5. Wait for wallet synchronization to complete

From Stronghold Backup

  1. Open IOTA Wallet and select "Restore Wallet"
  2. Choose "Restore from Backup File"
  3. Select your Stronghold backup file
  4. Enter your backup password
  5. Create a new wallet password
Tip: After restoring your wallet, verify that all accounts and balances have been recovered correctly before deleting any backups.

Security Best Practices

Essential security practices to keep your IOTA wallet safe.

Device Security

  • Keep your operating system and wallet software up to date
  • Use antivirus and anti-malware software
  • Enable device encryption (BitLocker, FileVault, etc.)
  • Lock your device when not in use
  • Avoid using public computers or WiFi for wallet access

Password Security

  • Use a unique, strong password for your wallet
  • Never reuse passwords from other services
  • Consider using a password manager
  • Change your password if you suspect it's compromised

Phishing Protection

Be vigilant against phishing attempts:

  • Always verify you're on the official IOTA website
  • Never click suspicious links in emails or messages
  • IOTA support will never ask for your seed phrase
  • Verify the authenticity of wallet software before installing
  • Use official download channels only

Transaction Verification

  • Always double-check recipient addresses before sending
  • Verify transaction amounts carefully
  • Start with small test transactions for new addresses
  • Monitor your transaction history regularly
Best Practice: For large holdings, consider using a hardware wallet for maximum security. IOTA Wallet supports Ledger integration.

Custom Nodes

Configure custom nodes for advanced network connectivity.

What are Nodes?

Nodes are servers that maintain the IOTA network and process transactions. By default, IOTA Wallet connects to official IOTA nodes, but you can configure custom nodes for:

  • Enhanced privacy
  • Better performance
  • Local development
  • Network redundancy

Adding a Custom Node

  1. Go to Settings > Node Configuration
  2. Click "Add Custom Node"
  3. Enter the node URL
  4. Optionally set authentication credentials
  5. Test the connection
  6. Save and activate the node

Node URL Format

Example Node URLs 
https://api.testnet.shimmer.network
https://chrysalis-nodes.iota.org
https://node.example.com:14265

Node Health Monitoring

IOTA Wallet automatically monitors node health and displays:

  • Connection status
  • Sync status
  • Response time
  • Available features
Warning: Only use trusted nodes. Malicious nodes could provide false information about your transactions.

Hardware Wallet Integration

Use a hardware wallet for maximum security.

Supported Hardware Wallets

IOTA Wallet currently supports:

  • Ledger Nano S
  • Ledger Nano S Plus
  • Ledger Nano X

Setting Up Ledger

  1. Install the IOTA app on your Ledger device using Ledger Live
  2. Connect your Ledger to your computer
  3. Open the IOTA app on your Ledger
  4. In IOTA Wallet, select "Connect Hardware Wallet"
  5. Follow the on-screen instructions
  6. Verify the connection on your Ledger display

Using Your Ledger

When using a hardware wallet:

  • Your private keys never leave the device
  • All transactions must be approved on the device
  • Always verify addresses on the device screen
  • Keep your Ledger firmware updated
Maximum Security: Hardware wallets provide the highest level of security for cryptocurrency storage. We recommend them for significant holdings.

API Integration

Integrate IOTA Wallet functionality into your applications.

IOTA SDK

For developers looking to integrate IOTA functionality, we provide comprehensive SDKs:

  • JavaScript/TypeScript SDK
  • Python SDK
  • Rust SDK
  • Java SDK

Installation

npm 
npm install @iota/sdk

Basic Usage Example

javascript 
const { Client } = require('@iota/sdk');

const client = new Client({
  nodes: ['https://api.testnet.shimmer.network'],
});

async function getInfo() {
  const info = await client.getInfo();
  console.log('Node Info:', info);
}

Common Issues

Solutions to frequently encountered problems.

Wallet Won't Open

Solution:

  1. Check that you're running the latest version
  2. Verify your system meets minimum requirements
  3. Try restarting your computer
  4. Reinstall the wallet if the problem persists

Transactions Pending for Long Time

Solution:

  • Check your network connection
  • Verify the node is synced and healthy
  • Try switching to a different node
  • Wait for network congestion to clear

Balance Not Showing

Solution:

  1. Wait for wallet synchronization to complete
  2. Check if you're connected to the correct network
  3. Try refreshing the wallet
  4. Verify the transaction on a block explorer

Cannot Connect to Node

Solution:

  • Check your internet connection
  • Verify firewall settings aren't blocking the wallet
  • Try using a different node
  • Disable VPN temporarily to test connection

Error Codes

Understanding wallet error messages.

Common Error Codes

Error Code Description Solution
ERR_001 Invalid password Re-enter your password or use recovery phrase
ERR_002 Node connection failed Check internet connection or switch nodes
ERR_003 Insufficient balance Add more funds to your wallet
ERR_004 Invalid address format Verify the recipient address is correct
ERR_005 Transaction timeout Retry the transaction or check node status
